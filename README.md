# ✦ Arcane Card Engine

A modular, browser-based **fantasy encounter & tarot-inspired card generator** designed for TTRPGs, VTT integration, worldbuilding, and narrative inspiration.

Generate structured encounters, story prompts, tarot archetypes, and thematic drivers instantly — with support for Major & Minor decks, reversals, theme switching, and expandable content tables.

---

## ✨ Features

### 🃏 Deck Modes

* **Hybrid Mode** – Narrative encounter majors + driver minors
* **Classic Tarot (22 Major Arcana)**
* **Tarot Suits (56 Minor Arcana)**
* **Mixed Mode** – Blended inspirational deck
* Optional **Reversals Toggle** (50% reversed interpretation)

---

### 🎭 Archetype System

Each card is built from structured narrative elements:

* **Axis** – Core thematic tension
* **Hook** – Immediate scenario seed
* **Tone**
* **Tier**
* **Archetype Category**

  * Combat
  * Social
  * Exploration
  * Extras (Clocks, Clues, Rewards)

---

### 🌌 Expanded Theme Engine

Switch themes to recontextualize card content:

* Classic Fantasy
* Grimdark
* High Magic
* Low Magic
* Mythic
* Swashbuckling
* Noir
* Desert
* Planar
* Steampunk
* Cyberpunk
* Sci-Fi

Each theme adjusts vocabulary, tone, and scenario framing.

## 🚀 Usage

1. Open `index.html` in your browser
2. Choose:

   * Major deck mode
   * Minor deck mode
   * Theme
   * Reversal toggle (optional)
3. Draw a card
4. Use the structured output to build encounters or narrative beats
